# GateKeeper6

Static MCEN/NIPR-focused safety scanner for Python repos (default: practical mode).

## Getting started

- Run a scan: `python3 -m mcen_scan /path/to/repo`
- Reports: `./mcen_audit/report.md` and `./mcen_audit/report.json`
- Specs: `docs/MCEN_Python_Safety_Scanner_SPEC.md`

## Development

- Tests: `python3 -m unittest -q`
